# 2026-04-15

## 修复 Post_setting.yaml YAML 解析错误

- **问题**：pytest 收集阶段报 `yaml.scanner.ScannerError: while scanning a simple key`，定位到 `test_case/UI/Test_Katana/All_YAML/Post/Post_setting.yaml` 第205行
- **根因**：`testT2705_VerifyGuest` 用例的步骤字段写成了 `test-step`（连字符），而其他用例统一使用 `test_step`（下划线），YAML 解析时 key 中的 `-` 触发了 scanner 错误
- **修复**：将 `test-step:` 改为 `test_step:`，YAML 解析恢复正常

## 修复 testT2705_VerifyPartner — check_default 定位问题

- **问题**：`check_default` 勾选 "9 people" 行的 default variant checkbox，但 XPath 未指定 index，默认命中行内第 0 个 checkbox（Exclude），而非 Default Variant 列
- **根因**：表格每行有两个 checkbox：index=0 是 Exclude，index=1 是 Default Variant
- **修复**：给 `check_exclude` 加 `index: 0`（明确），给 `check_default` 加 `index: 1`（关键修复）
